# Student Performance Analysis using SQL

##  Project Overview
This project analyzes student performance using SQL to identify average scores per teacher and detect high-risk students based on low attendance and low test scores.

##  Tools Used
- MySQL
- MySQL Workbench

##  Database Tables
- students
- attendance
- test_scores

##  Key Analysis
- Calculated average test scores per teacher
- Identified high-risk students with:
  - Attendance below 70%
  - Test score below 60

##  Business Insight
Teachers with lower average scores and higher numbers of high-risk students may require targeted academic interventions to improve student outcomes.
